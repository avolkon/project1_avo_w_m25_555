Игра "Лабиринт сокровищ"

Игрок блуждает по комнатам, собирает артефакты, разгадывает ребусы и охотится за главным призом - сокровищем. В игре есть несколько локаций, в каждой из которых что-то содержится: либо артефвкт, либо загадка, либо и то, и другое,
Важное условие, если в комнате есть нерешённая загадка - игрок не может взять артефакт до тех пор, пока не разгадает её.
Для получения главного приза нужно иметь 2 ключа.
Другие артефакты могут пригодиться в процессе игры и сослужить хорошую службу, рекомендуется посетить все комнаты и брать все, что встретится на пути.
Некоторые артефакты не являются обязательным условием для победы, но могут спасти при попадании в ловушку.
Перемещение по локациям происходит благодаря командам-направлениям.
Доступные для каждой локации направления, наличие загадок и артефактов выводятся в текстовом описании при попадании в локацию.
Игрок может совершать несколько видов действий - список всех доступных действий выводится по команде: помощь или help.

Доступные игроку действия и команды:
"go | идти <направление>   - идти (north, south, east, west)",
"north |  север | n        - идти на север",
"south |   юг   | s        - идти на юг",
"east  | восток | e        - идти на восток",
"west  |  запад | w        - идти на запад",    
"back  | назад             - вернуться назад", 
"take  | взять <item>      - взять предмет",
"use   | применить <item>  - использовать предмет",
"look  | осмотреть         - осмотреть комнату",
"items | артефакты         - показать артефакты",
"solve | решить            - решить загадку",
"help  | помощь            - помощь",
"exit  | выход             - выход из игры"

Для первого запуска игры необходимо выполнение следующих шагов:
1. Установите пакетный менеджер poetry (для этих целей лучше всего сразу воспользоваться поиском в интернете и пошаговым мануалом для вашей версии ОС, попытки зафиксировать 1 инструкцию для всех приводят к большим потерям времени на тестирование не всегда подходящих для вашей ОС алгоритмов).
Итог: через терминал необходимо установить poetry и проверить его версию (=наличие) командой:
poetry --version

После установки запусти команду 

poetry install

это нужно для последующего корректного запуска проекта и регистрации скриптов игры в виртуальном окружении

2. Для запуска игры в терминале нужно ввести:

make run

игра запустится.

Более длинный вариант с тем же итогом - запуском игры выглядит у меня так:

poetry run laby

в моей версии poetry слово project многократно используется как служебная команда -
хотелось избежать путаницы, поэтому - laby)

но в целом, make run для запуска должно быть вполне достаточно.
